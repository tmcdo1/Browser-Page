About Me

Howdy! I am a senior computer science student at Texas A&M University. I am the co-chair for workshops for the Aggie Coding Club, a project based organization that connects students to work on projects. We put on weekly workshops for our members covering topics from getting started with Git to creating a web application to using Docker to containerize an application.

I love learning about how to to automate the development lifecycle from using CI tools to deploying with an orchestration platform, like Kubernetes. Most of my experience is doing web development with some more experience in developing for Kubernetes and other development work.

Outside of development, I write a blog called Living Introspectively.
